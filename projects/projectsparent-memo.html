Parent Memo Notification System

Web-based communication system for schools to notify parents, confirm student arrivals, and manage parent records.

Project Screenshots

Key Features

  • Send real-time notifications to parents via email or SMS
  • Record and manage parent and student information
  • Confirm student arrivals during opening days
  • Generate detailed class-based and notification reports
  • Secure admin login and user-friendly dashboard

Technologies Used

  • Frontend: HTML, CSS, Bootstrap 5
  • Backend: PHP
  • Database: MySQL
  • Email Integration: Gmail SMTP (PHPMailer)
