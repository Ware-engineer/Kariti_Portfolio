Parent Memo Notification System

Technologies: PHP, MySQL, HTML, CSS, JavaScript, Email Integration

This project is a web-based system that enables school administrators to communicate efficiently with parents. The platform allows sending of email notifications, confirmation of student arrivals, management of parent records, and generation of reports through an intuitive dashboard.

Key Features

  • Send notifications with subject lines and messages to parents.
  • Confirm student arrivals by class and notify parents automatically.
  • View, edit, and manage parent records in the system.
  • Export reports in PDF or Excel formats.
  • Real-time email delivery using Gmail app password integration.
